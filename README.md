# 📌 ACEest Fitness & Gym – DevOps CI/CD Pipeline Project
> ✨ This project demonstrates the implementation of modern DevOps practices for the ACEest Fitness & Gym management system. The application is developed using Python and provides functionality for managing gym clients, workouts, and progress tracking.The project integrates DevOps tools such as Git, Docker, Jenkins, and GitHub Actions to automate the build, testing, and deployment workflow. The objective is to ensure code integrity, consistent environments, and reliable automated builds.

  ## 📌 About
Python Project that helps to journal your fitness routines, by logging your Fitness workout and your duration for each workout. 
---
## 🛠 Tech Stack
- Python
- Pytest (Unit Testing)
- Docker (Containerization)
- Jenkins (Build Automation)
- Git & GitHub (Version Control)

## 📦 Installation
# Clone the repo
git clone https://github.com/2024ht66055/ACEest-Fitness-Gym.git
# Navigate into the project
cd ACEest-Fitness-Gym

# Install dependencies
pip install -r requirements.txt
```
---
## 🚀 Usage
```
python src/aceest_fitness.py 
```
The ACEest Fitness application interface will open, allowing the user to manage gym clients, workouts, and progress tracking.

Application is accessible via http://127.0.0.1:5000/
